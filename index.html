2. React Fiber의
두 가지 개념

1) Fiber Architecture

리액트가 컴포넌트를 렌더링하고 UI를 업데이트하는 방식을 완전히 새롭게 디자인한 아키텍처

2) Fiber Node

리액트 트리 구조의 각 노드를 나타내는 객체

3. React Fiber Node란?

React Fiber Node

: 하나의 작업 단위

  • 하나의 컴포넌트에는 하나의 파이버 노드가 존재
  • 리액트 내부에서 관리하는 가상 돔의 노드 객체
  • 리액트 요소를 확장
